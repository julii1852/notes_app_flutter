# Notes App Flutter

## Описание
Мобильное приложение для создания, редактирования, удаления и поиска заметок с хранением данных в SQLite.

## Функции
- Создание заметок (заголовок, содержание, дата/время)
- Редактирование и удаление
- Поиск по ключевым словам
- Локальное хранение данных через SQLite
